## API Reference

### TelInput

TelInput component extends the props from react-phone-number-input with some customizations.

| Prop                         | Type                                   | Default     | Description                                             |
| ---------------------------- | -------------------------------------- | ----------- | ------------------------------------------------------- |
| `value`                      | `E.164 string`                         | `undefined` | Phone number in E.164 format                            |
| `onChange`                   | `(value: string \| undefined) => void` | `undefined` | Called when phone number value changes                  |
| `defaultCountry`             | `string`                               | `undefined` | Two-letter country code for initial selection           |
| `international`              | `boolean`                              | `false`     | Whether to show the international format                |
| `countryCallingCodeEditable` | `boolean`                              | `true`      | Whether the country calling code part is editable       |
| `disabled`                   | `boolean`                              | `false`     | Disables both the country select and phone number input |
| `className`                  | `string`                               | `""`        | Additional CSS class for the component                  |
| `placeholder`                | `string`                               | `undefined` | Placeholder for the phone number input                  |
| `...rest`                    | `PhoneNumberInput.Props`               | -           | All other props from react-phone-number-input           |

## Accessibility

- The country select dropdown is fully keyboard navigable
- The component uses proper ARIA attributes for improved screen reader support
- The flag icons have appropriate alt text for accessibility
