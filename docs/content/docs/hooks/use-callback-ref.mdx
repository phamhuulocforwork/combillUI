## Usage

```tsx
import { useCallbackRef } from "@/hooks/use-callback-ref";

const callback = useCallbackRef((value) => {
  // Handle callback here
});
```

This hook:

- Takes a callback function as input
- Returns a stable callback function
- Prevents unnecessary re-renders when the callback changes
- Useful for cases where callbacks need to be passed as props
