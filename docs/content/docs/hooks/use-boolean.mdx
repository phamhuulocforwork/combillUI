## Usage

```tsx
// Initialize with default value (false)
const { value, toggle, setTrue, setFalse } = useBoolean();
//Or with a specific initial value
const { value, toggle, setTrue, setFalse } = useBoolean(true);
```

The hook provides these utilities:

- `value`: Current boolean state
- `setValue`: Standard React state setter
- `setTrue`: Set state to true
- `setFalse`: Set state to false
- `toggle`: Toggle between true and false
